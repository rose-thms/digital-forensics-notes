# Magnet AXIOM

## Overview

Magnet AXIOM is a digital forensic investigation platform used for acquiring, processing, analyzing, and reporting digital evidence from computers, mobile devices, and cloud sources.

It helps investigators identify artifacts related to user activity, application usage, communication, and system behavior.

---

# AXIOM Workflow

The typical Magnet AXIOM workflow consists of:

1. Evidence Acquisition
2. Evidence Processing
3. Artifact Analysis
4. Review and Validation
5. Report Generation

---

# 1. Evidence Sources

AXIOM supports analysis of:

## Computer Evidence

Examples:

- Hard disk images
- SSD images
- USB drives
- Logical collections

Supported formats:

- E01
- RAW/DD
- AFF
- AD1


## Mobile Evidence

Sources include:

- Physical extractions
- Logical extractions
- File system extractions
- UFDR files

---

# 2. AXIOM Process

During processing, AXIOM:

- Parses forensic artifacts
- Extracts metadata
- Recovers deleted data
- Identifies application activity
- Creates searchable results

---

# 3. Important Artifacts

## Browser Artifacts

Can reveal:

- Browsing history
- Downloads
- Cookies
- Search activity


## Communication Artifacts

Examples:

- Messages
- Emails
- Chat applications


## User Activity

Includes:

- Recently opened files
- Login information
- Application usage


## Multimedia Artifacts

Includes:

- Images
- Videos
- Metadata
- EXIF information

# 4. Artifact Categories

Common AXIOM artifact categories:

## Internet Activity

## Application Data

- Messaging applications

## System Information

- OS details

## File System

- Deleted files
- Metadata
- File timestamps

---

# 5. File System and Carving

AXIOM uses:

## Parsing

Extracts known artifacts from supported applications and file structures.

## File Carving

Recovers files based on known file signatures.

Useful for:

- Deleted files
- Damaged file systems
- Unallocated space analysis
